# Data Analyst Project Starter 🚀

Скрипт для мгновенного развертывания правильной структуры проекта по анализу данных. 
Создан для того, чтобы не тратить время на рутину и сразу переходить к инсайтам.

## Что внутри?
Скрипт создает логичную архитектуру, разделяя код, данные и тесты:
- `src/utils/` — твоя личная библиотека (конвертация дат, очистка текста, форматирование Excel).
- `scripts/` — место для основных скриптов анализа.
- `data/` — раздельные папки для сырых (`raw`) и обработанных (`processed`) данных.
- `tests/` — папка для проверки твоих функций.
- `.gitignore` — уже настроен под аналитиков (игнорирует тяжелые .csv, .xlsx и кэш).

## Как использовать
1. Скопируйте файл `generator.py` в папку вашего будущего проекта.
2. Запустите его:
   ```bash
   python generator.py
   ```
3. Структура готова! Начинайте импортировать свои утилиты:
   ```python
   from src.utils.time_utils import to_datetime
   ```

## 🛠 Почему это полезно?
Многие аналитики хранят всё в одной куче. Этот шаблон помогает:
1. Соблюдать чистоту кода (DRY — Don't Repeat Yourself).
2. Безопасно работать с Git, не выгружая лишние данные.
3. Легко переносить свои наработки (utils) из проекта в проект.
